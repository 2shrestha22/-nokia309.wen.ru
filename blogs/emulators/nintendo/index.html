NES (Nintendo Entertainment System) Emulator-Nescube

(touch not supported-only for keypad mobiles)

Nescube is an emulator for nes rom. You can play Super Mario, Contra and other classic video games using this emulator. Rom is game file which is emulated by Emulator (Nescube).

This is hacked/full version of Nescube.

Benifits of hacked/full version:
# This is color version where demo version is black and white.
# You can save running game anywhere and any time and load/resume it later from same point so you don't lose game progress and levels.
# It can load roms/games from memory card where previous version can't load from memory card directly so you don't need to repack emulator with roms.
# You can save multiple games at a time.
